  Essential Tier (Install these first)

  1. Telescope - Fuzzy finder for everything
  - Find files, grep text, search commands, buffers, git commits
  - Replaces the need for file explorers in many workflows
  - Most used plugin by far

  2. nvim-treesitter - Syntax understanding
  - Vastly superior syntax highlighting
  - Enables smart text objects, refactoring, code navigation
  - Foundation for many other plugins

  3. LSP + mason.nvim - Language servers
  - Autocomplete, go-to-definition, diagnostics, hover docs
  - mason.nvim makes installing language servers trivial
  - Transforms Neovim into a true IDE

  4. nvim-cmp - Autocompletion engine
  - Works with LSP, buffers, paths, snippets
  - Highly configurable sources
  - Essential for productive coding

  High Impact Tier

  5. which-key.nvim - Keybinding discovery
  - Shows popup of available keybindings as you type
  - Eliminates the need to memorize everything
  - Life-changing for learning and organization

  6. gitsigns.nvim - Git integration
  - Shows git changes in sign column
  - Stage/unstage hunks, blame, diff views
  - Essential for git workflows

  7. Comment.nvim - Smart commenting
  - gcc to comment line, gc in visual mode
  - Language-aware, integrates with treesitter
  - Simple but transforms editing speed

  8. nvim-autopairs - Auto-close pairs
  - Auto-close brackets, quotes, tags
  - Smart deletion and jumping
  - Quality of life essential

  Navigation Enhancement

  9. flash.nvim (or hop.nvim/leap.nvim) - Quick jumps
  - Jump to any visible location with 2-3 keystrokes
  - Changes how you move through code
  - Once you use it, you can't go back

  10. harpoon - File marks
  - Quick toggle between your main 4-5 files
  - Alternative to tabs/buffers for active files
  - Beloved by many power users

  Quality of Life

  11. lualine.nvim - Beautiful statusline
  - Shows mode, git branch, diagnostics, file info
  - Highly customizable
  - Makes Neovim feel polished

  12. indent-blankline.nvim - Indentation guides
  - Visual guides for indentation levels
  - Highlights current scope
  - Improves readability significantly

  13. trouble.nvim - Better diagnostics
  - Prettier list of errors/warnings/TODOs
  - Quick navigation to issues
  - Better than quickfix/location list

  14. nvim-surround - Surround operations
  - Change "hello" to 'hello' with cs"'
  - Add/delete/change surrounding pairs
  - Powerful text editing primitive

  15. undotree - Visualize undo history
  - See your entire undo tree
  - Time-travel through changes
  - Saves you from accidental data loss

  16. toggleterm.nvim - Better terminals
  - Floating or split terminals
  - Persistent terminal sessions
  - Much better than :terminal
